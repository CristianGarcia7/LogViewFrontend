## Superficies (jerarquía de profundidad)

```
level_0: bg #000000  /  text #FFFFFF
level_1: bg #080808  /  text #F5F5F5   ← canvas base de la app
level_2: bg #0D0D0D  /  text #DCDCDC   ← card background
level_3: bg #131313  /  text #8C8C8C   ← tooltip, dropdown item bg
level_4: bg #191919  /  text #6E6E6E   ← card gradient light
level_5: bg #202020  /  text #545454   ← card gradient dark / disabled bg
level_6: bg #3B3B3B  /  text #3B3B3B   ← dividers sutiles
```
